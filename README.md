# riffr <img src="https://storage.googleapis.com/pr-newsroom-wp/1/2023/05/Spotify_Primary_Logo_RGB_Green.png" width="50" height = "50" title="logo">

#### riffr is a full-stack web application that makes finding concerts of your favorite artists easier and more centralized than ever. 

## How it works
* Prompts the user to login via Spotify OAuth
<img width="600" alt="s1" src="https://github.com/user-attachments/assets/f0f37734-1196-4d63-b55f-d4323b1d592e">

* Asks the user for a number between 1-50
<img width="600" alt="s2" src="https://github.com/user-attachments/assets/449b95f4-31ef-4aad-9da4-e4edadb42b4a">

* Retrieves the user's N-most listened artists using the Spotipy API
<img width="600" alt="s3" src="https://github.com/user-attachments/assets/9cbdf54e-d790-4eec-90dc-2676aa311375">

* Utilizes Ticketmaster's API to filter countries by eligible country
<img width="600" alt="s4" src="https://github.com/user-attachments/assets/dfa3f2ab-c26d-4af1-bfc6-30225e026f1f">

* Allows a user to view upcoming concert information for all top artists
<img width="600" alt="s5" src="https://github.com/user-attachments/assets/995c331f-1226-4ba1-ba09-73f8f2237926">

* Leverages Google Maps API to show satellite imaging of venues and surrounding areas
<img width="600" alt="s6" src="https://github.com/user-attachments/assets/a9f15e48-146d-45df-b1cf-00c32cc4bef8">

* Provides direct links to Ticketmaster to buy tickets for each upcoming concert
* Displays additional concert info such as venue address, city, time, date, and more
<img width="600" alt="s7" src="https://github.com/user-attachments/assets/d5095821-37b8-4029-b6e5-7ad7206b4215">
